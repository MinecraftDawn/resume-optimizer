# LinkedIn Output Format

## 輸出報告格式

### LinkedIn 報告模板

**報告開頭固定用以下 blockquote 摘要卡片（必填）：**

```
> ## 💼 LinkedIn 個人頁面評分報告
> [若有舊版報告，標題後加：「（第 N 次分析）」]
>
> **總分　** ███████░░░ **X / 100 分**
>
> 💪 **最強亮點：** [一句話點出最突出的區塊]
> 🎯 **最需改善：** [一句話點出最需優先處理的弱點]
> 📌 **分析目標：** [主動求職 / 被動觀望 / 個人品牌建立]
```

**摘要卡片之後，輸出 2–3 句整體印象（總覽），特別說明個人品牌清晰度。**

### 詳細建議

對每個達成率 <75% 的區塊，依以下格式輸出：

**❌ 嚴重不足（blockquote 強調）：**
```
> #### ❌ [區塊名稱]（X / X 分，XX%）
>
> **問題核心：** [一句話說明最關鍵的缺失]
>
> [具體建議，工作經歷和 About 必含 XYZ 格式改寫範例]
```

**⚠️ 一般建議：**
```
#### ⚠️ [區塊名稱]（X / X 分，XX%）

[具體建議]
```

若有 JD → 在詳細建議末尾加入 LinkedIn JD Gap 分析表。

---

### 行動清單

使用 checkbox 格式：

```
### ✅ 下一步行動清單（今日優先執行）

- [ ] **[最高影響力：通常是 Headline 或 About 改寫]**（預估 +X 分）
  → [附改寫範例或具體行動]
- [ ] **[次高影響力：通常是工作經歷補量化成就]**（預估 +X 分）
  → [具體行動]
- [ ] **[第三：技能 endorsement 或 Featured 補充]**（預估 +X 分）
  → [具體行動]
[若雙平台模式，追加：]
- [ ] **雙平台一致性確認**
  → 核對 104 與 LinkedIn 的成就數字、職稱時間軸
```

**狀態圖示（同 104 標準）：**
| 圖示 | 意義 | 達成率 |
|------|------|--------|
| ✅ | 優秀 | ≥75% |
| ⚠️ | 尚可，建議改善 | 50–74% |
| ❌ | 嚴重不足，優先處理 | <50% |

---

### 雙平台增量分析模式輸出

若用戶選擇「雙平台」，LinkedIn 報告在 104 報告之後輸出，並在最末加入：

```
### 雙平台一致性提醒
✅ 需確認一致：成就數字（104 與 LinkedIn 描述的同一成果數字應相同）
✅ 需確認一致：工作年資與職稱的起訖時間
⚠️ 允許差異：語氣（104 中性 → LinkedIn 第一人稱）
⚠️ 允許差異：詳細程度（LinkedIn About 可更故事性，104 自傳更精簡）
❌ 應避免：104 說「提升 30%」但 LinkedIn 說「提升 50%」（同一成就數字不一致）
```
