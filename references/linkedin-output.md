# LinkedIn Output Format

## 輸出報告格式

### LinkedIn 報告模板

```
## LinkedIn 個人頁面評分報告
[若有舊版報告，標題後加：「（第 N 次分析）」]

### 總覽
[2–3 句整體印象，點出最強亮點與最需改善的區塊。特別說明個人品牌清晰度。]

### 各區塊評分

> 分析目標：[主動求職 / 被動觀望 / 個人品牌建立]

| 區塊 | 得分 | 滿分 | 達成率 | 狀態 |
|------|------|------|--------|------|
| Headline | X | 10 | XX% | ✅/⚠️/❌ |
| About（關於） | X | 20 | XX% | |
| 工作經歷 | X | 30 | XX% | |
| 技能 | X | 15 | XX% | |
| 教育背景 | X | 8 | XX% | |
| Featured（精選） | X | 7 | XX% | |
| 專案 | X | 5 | XX% | |
| 推薦文 | X | 3 | XX% | |
| 語言／證照 | X | 2 | XX% | |
| **總分** | **X** | **100** | **XX%** | |

### 詳細建議
[對達成率 <75% 的各區塊提供建議，依影響力由高到低排列]
[工作經歷和 About 必含 XYZ 格式改寫範例]

[若有 JD → 加入 LinkedIn JD Gap 分析表]

### 下一步行動清單（今日優先執行）
1. [最高影響力：通常是 Headline 或 About 改寫，附改寫範例]
2. [次高影響力：通常是工作經歷補量化成就]
3. [第三：技能 endorsement 或 Featured 補充]
[若雙平台模式，加第 4 條：雙平台一致性確認]
```

**狀態圖示（同 104 標準）：**
| 圖示 | 意義 | 達成率 |
|------|------|--------|
| ✅ | 優秀 | ≥75% |
| ⚠️ | 尚可，建議改善 | 50–74% |
| ❌ | 嚴重不足，優先處理 | <50% |

---

### 雙平台增量分析模式輸出

若用戶選擇「雙平台」，LinkedIn 報告在 104 報告之後輸出，並在最末加入：

```
### 雙平台一致性提醒
✅ 需確認一致：成就數字（104 與 LinkedIn 描述的同一成果數字應相同）
✅ 需確認一致：工作年資與職稱的起訖時間
⚠️ 允許差異：語氣（104 中性 → LinkedIn 第一人稱）
⚠️ 允許差異：詳細程度（LinkedIn About 可更故事性，104 自傳更精簡）
❌ 應避免：104 說「提升 30%」但 LinkedIn 說「提升 50%」（同一成就數字不一致）
```
