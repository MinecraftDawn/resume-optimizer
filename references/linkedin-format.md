# LinkedIn Format

## 欄位清單與充足標準

執行 LinkedIn Missing Section Audit 時，依下表逐欄檢查。充足度判斷以「有效內容是否達到充足門檻」為準，不是「有沒有填滿上限」。

| 欄位 | 必填/選填 | 充足標準 | 缺失處理 |
|------|---------|---------|---------|
| **Headline（標題）** | 必填 | 包含職稱 + 價值主張，非只有公司名稱，≤120 字元 | ❌ 嚴重缺口 |
| **About（關於）** | 強烈建議 | 200 字以上，有開場鉤子 + 核心成就 + 意向 | ⚠️ 若 <100 字或空白 |
| **工作經歷** | 必填 | 每個職位有描述，至少 1 條含數字的成就 | ❌ 若全無描述 |
| **工作經歷 — 技能標籤** | 建議 | 每個職位 ≥3 個技能標籤 | ⚠️ 若缺失 |
| **工作經歷 — 廣播（媒體附件）** | 選填 | 有作品/簡報更佳，但非必要 | ⚪ |
| **教育背景** | 必填 | 學校 + 學位 + 科系 | ⚠️ 若只有學校名 |
| **教育背景 — 說明** | 選填 | 若有重要課程/社團/論文，建議填寫 | ⚪ |
| **專案** | 選填（技術/創意職位強烈建議）| 有說明 + 技能標籤 + 時間區間 | ⚪（技術/創意職 → ⚠️）|
| **技能** | 強烈建議 | ≥10 個技能，核心技能有 endorsement | ⚠️ 若 <5 個 |
| **榮譽獎項** | 選填 | 有則填，附授獎機構 + 日期 + 說明 | ⚪ |
| **測驗成績（如語言考試）** | 選填（外商/特定職位重要）| 填入分數 + 日期 + 說明 | ⚪ |
| **語言能力** | 建議 | 至少填母語 + 英文流利程度 | ⚠️ 若完全缺失 |
| **執照或證照** | 視職位而定 | 有發照機構 + 發照日期（+到期日若適用）| ⚪（特定職缺 → ❌）|
| **Featured（精選）** | 強烈建議 | 置頂 ≥1 個最強成就、文章或作品，且與職業強相關 | ⚠️ 若空白 |
| **推薦文（Recommendations）** | 選填 | ≥2 則他人為你撰寫的推薦文 | ⚪（資深者 → ⚠️）|
| **職場意向（Open to Work）** | 視求職狀態而定 | 主動求職者建議開啟 | ⚪ |

**缺失處理圖示：**
- ❌ **嚴重缺口：** 對填寫結果影響最大，列為最高優先建議
- ⚠️ **內容不足：** 達成率低，提供具體補充建議
- ⚪ **可補充：** 若有則更佳，不強迫填寫
- ❓ **無法評估：** 用戶未提供足夠資訊，提示補充
